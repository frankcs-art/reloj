## 2025-02-12 - Reset to Defaults UX Pattern
**Learning:** In complex configuration tools, providing a prominent "Reset to Defaults" button encourages user experimentation and reduces "fear of breaking" the UI by offering a safe path back to a known-good state.
**Action:** Implement a Reset button alongside primary actions in configuration panels, using abstract variable names for configurable properties to maintain semantic clarity.
